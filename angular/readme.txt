1.Open index.html file
2.Click on “Login” button. This will call the login api which will return the authId
for use as authentication token in header of subsequent api calls
3.Click on “GetHeaders” button to get the column header data
4.Click on “GetData” to get the grid data

Note : The index.html is for demo purpose only and it’s use is optional. You can directly use the angular code in the app.js file.
